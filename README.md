## 2. 必要なソフトウェアのダウンロード手順
以下のリンクより、それぞれのインストーラーを入手すること。

1. **GNU Octave 9.4:**
   `octave-9.4.0-w64-installer.exe` をダウンロードする。
   https://mirrors.ustc.edu.cn/gnu/octave/windows

2. **Dynare 6.3:**
   `dynare-6.3-win.exe` をダウンロードする。
   https://www.dynare.org/release/windows/

3. **Occbin_update:**
   以下のページで **Code** タブを選択し、緑の **Code** ボタンから **Download ZIP** を選択する。
   https://github.com/JohannesPfeifer/Occbin_update


## 3. シミュレーションコードの入手
修論のシミュレーションコード一式をダウンロードする。

**masters_thesis:**
以下のページで **Code** タブを選択し、緑の **Code** ボタンから **Download ZIP** を選択する。
https://github.com/nanazou/masters_thesis


## 4. 実行手順
以下の手順でシミュレーションを実行する。

1. **Octave の起動:**
   Octave を開き、左側のファイルブラウザを使用して、本シミュレーションコード一式（env_setup.m があるフォルダ）へ移動する。

2. **環境設定の実行:**
   **注意：** 必ず **env_setup.m** が存在するフォルダへ移動してから以下のコマンドを実行すること。
   コマンドウィンドウで以下を入力する。
   ```
   env_setup
   ```

3. **プロジェクトフォルダへの移動:**
   ファイルブラウザから、実行したいプロジェクトフォルダ（例：clt_project）へ移動する。

4. **メインプログラムの実行:**
   コマンドウィンドウで以下を入力する。
   ```
   clt_main
   ```


## 5. 結果の確認とカスタマイズ
- シミュレーション結果（グラフ PDF、データ .mat/.csv）は **results** フォルダ内に出力される。
- 初期状態ですでに実行結果が入っている。
- 政策パラメータやショックの大きさ変更してシミュレーションしたい場合は **clt_main.m** の該当箇所を書き換えた後に実行する。
